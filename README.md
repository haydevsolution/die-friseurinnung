# Die Friseurinnung Stuttgart · Ludwigsburg

Moderne, responsive One-Page-Website für die Friseurinnung Stuttgart | Ludwigsburg.
Neugestaltung der bestehenden Seite [die-friseurinnung.de](https://www.die-friseurinnung.de/).

## Stack

Reines, statisches Frontend — kein Build-Schritt nötig:

- **HTML5** (`index.html`)
- **CSS3** (`styles.css`) — editoriales Design, warmes Ink + Roségold-Akzent, voll responsive
- **Vanilla JavaScript** (`script.js`) — Scroll-Reveal, mobiles Menü, Scroll-Fortschritt
- Google Fonts: *Cormorant Garamond* & *Inter*

## Inhalte

- **Die Innung** — Vertretung, Gemeinschaft, Qualität
- **Ausbildung · Karriere · Fortbildung**
- **Kontakt** — Standorte Stuttgart & Ludwigsburg mit Öffnungszeiten

## Lokal ansehen

Einfach `index.html` im Browser öffnen, oder ein lokaler Server:

```bash
python3 -m http.server 8000
# http://localhost:8000
```

## Deployment (GitHub Pages)

Die Seite wird über **GitHub Pages** vom `main`-Branch (Root) ausgeliefert.
Settings → Pages → Branch: `main` / `/ (root)`.
